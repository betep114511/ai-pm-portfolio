# SafetyLens PRD

## 1. 背景

实验室和化工仓储场景需要长期管理 SDS、危险化学品台账、储存规则和 PPE 要求。SDS 文件篇幅长、格式不统一，人工阅读容易遗漏关键风险。通用大模型可以帮助阅读文档，但危化品安全不能完全依赖模型生成，需要可追溯、可审核、可留痕的产品机制。

SafetyLens 的目标是：用 AI 提升 SDS 审核效率，同时让高风险判断保留人工确认和审计记录。

## 2. 目标用户

- EHS 专员：需要审核化学品风险、储存条件和合规记录。
- 实验室管理员：需要确认试剂储存、PPE、废液和培训要求。
- 仓储/采购人员：需要判断新采购化学品是否可入库、怎么储存。
- 研发新人：需要快速理解化学品操作风险。

## 3. 用户痛点

| 痛点 | 表现 | 影响 |
| --- | --- | --- |
| SDS 阅读慢 | 文件长、字段分散、格式不同 | 审核耗时高 |
| 风险容易漏 | GHS、PPE、不相容物质分散 | 安全隐患 |
| 规则难记 | 酸碱、氧化剂、易燃品、毒性物质规则复杂 | 新人依赖人工 |
| 审核不可追溯 | AI 或人工修改没有记录 | 合规风险 |
| 误报多 | 规则过粗会导致大量无效警报 | 用户不信任 |

## 4. 产品目标

### 业务目标

- 将单份 SDS 初审时间降低 50%。
- 提升高风险项召回率，降低漏报。
- 形成审核记录，支持后续安全检查。

### 产品目标

- 3 分钟内完成 SDS 上传、结构化抽取和风险摘要。
- 高风险项必须展示来源字段和人工确认状态。
- 用户能生成可导出的安全检查清单。

## 5. MVP 范围

### P0 功能

1. SDS 上传与解析
   - 支持 PDF/文本。
   - 提取化学品名称、CAS、GHS 象形图、危险说明、PPE、储存条件、急救措施。

2. 风险摘要
   - 按易燃、腐蚀、毒性、氧化性、环境危害等维度分级。
   - 展示证据来源和置信度。

3. 规则校验
   - 不相容物质校验。
   - 储存条件校验。
   - PPE 完整性校验。

4. 人工审核
   - 高风险项必须人工确认。
   - 审核人可修改 AI 抽取字段。
   - 保留修改前后记录。

5. 安全清单
   - 生成储存、操作、应急、废弃处理 checklist。

### P1 功能

- 批量 SDS 审核。
- 化学品库风险地图。
- 过期 SDS 提醒。
- 与库存台账联动。

### P2 功能

- 企业权限与审批流。
- 多语言 SDS 对照。
- EHS 培训材料自动生成。

## 6. 核心流程

### 流程 A：新化学品入库审核

1. 采购/管理员上传 SDS。
2. 系统抽取 CAS、危险类别和储存条件。
3. 规则引擎检查是否与现有库位冲突。
4. 高风险项进入 EHS 人工审核。
5. 生成入库建议和安全清单。

### 流程 B：实验前安全确认

1. 用户搜索化学品。
2. 查看风险摘要、PPE 和操作注意事项。
3. 系统提示不相容物质和废液要求。
4. 用户导出实验前 checklist。

### 流程 C：AI 抽取纠错

1. EHS 发现某字段错误。
2. 修改字段并选择错误类型。
3. 系统记录 AI 原始输出、人工修改和审核人。
4. 样本进入评测集。

## 7. AI 方案

### LLM 负责

- 非结构化 SDS 文档理解。
- 字段抽取。
- 风险摘要草稿。
- checklist 文案生成。

### 规则引擎负责

- 不相容物质判断。
- 强制储存规则。
- PPE 缺失检查。
- 高风险拦截和人工审核触发。

### 人工负责

- 高风险项最终确认。
- 规则冲突处理。
- 审计和合规签字。

## 8. 输出结构

```text
化学品摘要：名称、CAS、用途、危险等级。
危险类别：GHS 分类、危险说明、证据来源。
储存建议：温度、通风、库位、不相容物。
PPE 要求：手套、护目镜、实验服、呼吸防护。
应急措施：接触、吸入、泄漏、火灾。
审核状态：AI 草稿 / 待人工确认 / 已确认。
```

## 9. 风险控制

| 风险 | 策略 |
| --- | --- |
| LLM 抽取错误 | 字段级置信度 + 人工审核 |
| 高风险漏报 | 规则引擎兜底 + 高风险召回优先 |
| 误报过多 | 按风险等级分层展示，支持人工标记 |
| 审核责任不清 | 审计日志记录 AI 输出和人工修改 |
| SDS 版本过期 | 版本号、发布日期和过期提醒 |

## 10. 指标

### 北极星指标

每周完成并通过人工确认的高风险 SDS 审核数。

### AI 质量指标

- 字段抽取准确率。
- 高风险项召回率。
- 不相容物质识别召回率。
- 误报率。
- 人工修改率。

### 产品指标

- 单份 SDS 初审耗时。
- 审核通过率。
- 清单导出次数。
- 人工审核完成率。
- 用户满意度。

## 11. 上线门禁

MVP 试点前必须满足：

- CAS/名称抽取准确率 >= 98%。
- 高风险项召回率 >= 95%。
- 不相容物质召回率 >= 95%。
- 高风险漏报率 = 0 作为上线目标。
- 所有高风险项均进入人工审核。

## 12. 项目边界

SafetyLens 不是自动合规判定系统。它提供初审、证据整理和风险提示，最终安全责任由具备资质的 EHS 或实验室负责人确认。
