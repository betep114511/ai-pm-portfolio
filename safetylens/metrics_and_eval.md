# SafetyLens 指标与评测方案

## 1. 评测目标

SafetyLens 的评测重点不是生成文案是否自然，而是：

- SDS 关键字段是否抽取正确。
- 高风险项是否被召回。
- 不相容物质是否被识别。
- 人工审核是否能发现和修正错误。
- 产品是否能留下可审计记录。

## 2. 北极星指标

每周完成并通过人工确认的高风险 SDS 审核数。

原因：

- 既体现使用量，也体现高价值场景。
- 避免只追求上传量，而忽视审核质量。

## 3. 核心指标

| 指标 | 定义 | MVP 目标 |
| --- | --- | --- |
| CAS 抽取准确率 | CAS 字段完全正确的比例 | >= 98% |
| 名称抽取准确率 | 中文/英文名称正确比例 | >= 98% |
| GHS 分类准确率 | 危险类别识别正确比例 | >= 90% |
| PPE 召回率 | 必要 PPE 是否完整召回 | >= 90% |
| 高风险项召回率 | 高风险信息是否被识别 | >= 95% |
| 不相容物质召回率 | 不相容物质是否被识别 | >= 95% |
| 高风险漏报率 | 高风险未提示比例 | 目标为 0 |
| 误报率 | 错误标记风险的比例 | <= 15% |
| 人工修改率 | AI 字段被修改比例 | 用于定位薄弱字段 |

## 4. 评测集结构

每条样本包含：

- 化学品名称。
- CAS。
- SDS 片段。
- 标准危险类别。
- 标准 PPE。
- 标准储存条件。
- 不相容物质。
- 期望审核行为。
- 风险等级。

## 5. 样本覆盖

| 类型 | 占比 | 示例 |
| --- | --- | --- |
| 易燃液体 | 25% | 乙醚、丙酮、乙醇 |
| 腐蚀性试剂 | 20% | 盐酸、硫酸、氢氧化钠 |
| 氧化剂 | 20% | 硝酸、过氧化氢、高锰酸钾 |
| 毒性/健康危害 | 20% | DMF、苯、甲醛 |
| 特殊反应风险 | 15% | 氢化钠、金属钠、过氧化物 |

## 6. 评测方法

### 自动评测

- 正则/规则校验 CAS。
- 字段级 exact match 或同义词匹配。
- GHS 分类集合对比。
- PPE checklist 覆盖率。
- 不相容物质规则召回。

### LLM-as-Judge

用于辅助判断：

- 风险摘要是否忠实于 SDS。
- 是否遗漏关键危害。
- 是否加入无来源安全建议。

### 人工评测

- 高风险样本 100% 人审。
- 中风险样本 30% 抽检。
- 低风险样本 10% 抽检。

## 7. 上线门禁

上线前必须满足：

- 高风险漏报率为 0。
- 高风险项 100% 进入人工审核。
- CAS/名称抽取准确率不低于 98%。
- 不相容物质召回率不低于 95%。
- 所有自动结论有 SDS 来源或规则来源。

## 8. 失败归因

| 错误类型 | 说明 | 修复方式 |
| --- | --- | --- |
| 字段抽取失败 | CAS、名称、PPE 错漏 | 优化 prompt、OCR、字段校验 |
| 危险分类遗漏 | GHS 分类没识别 | 补充训练样本和分类规则 |
| 不相容漏报 | 规则库缺失或匹配失败 | 补规则、补同义词 |
| 误报过多 | 风险触发太宽 | 调整阈值和规则条件 |
| 来源不足 | 结论没有 SDS 依据 | 强制来源字段 |

## 9. 面试表达

> 我会把 SafetyLens 当作高风险 AI 产品来评估。LLM 的作用是抽取和整理，规则引擎负责明确约束，高风险最终要人审。指标上我重点看高风险召回率和漏报率，而不是只看生成质量，因为安全场景最怕漏掉真正危险的内容。
