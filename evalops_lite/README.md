# EvalOps Lite 作品集项目

EvalOps Lite 是一个轻量级 LLM 应用质量评估与迭代看板，面向 AI 产品经理、Prompt 工程师和算法/工程团队。它支持测试集管理、Prompt/模型/RAG 参数对比、自动评测、人工抽检、上线门禁和失败案例回流。

## 项目定位

这个项目用于证明你理解 AI 产品上线不是“试几条 prompt”，而是需要可重复评测、失败归因和质量门禁。

核心能力：

- 设计 LLM 应用评测指标。
- 管理测试集和回归样本。
- 对比 Prompt、模型和检索参数。
- 处理质量、风险、成本、延迟之间的 trade-off。

## 文件说明

- `prd.md`：完整 PRD。
- `user_research.md`：用户画像、访谈问题和需求洞察。
- `competitive_analysis.md`：竞品与替代方案分析。
- `case_study.md`：一页项目复盘。
- `metrics_and_eval.md`：评测体系和上线门禁。
- `presentation_guide.md`：简历和面试展示指南。
- `interview_script.md`：面试讲述稿。
- `data/eval_cases.csv`：样例评测集。
- `prototype/index.html`：静态交互原型。
