# InterviewMate AI 指标与评测

## 北极星指标

每周完成并获得有效反馈的模拟面试次数。

## 产品指标

| 指标 | 定义 |
| --- | --- |
| 首次练习完成率 | 用户完成一轮主问题 + 追问 + 反馈 |
| 7 日留存 | 用户是否持续练习 |
| 平均追问轮次 | 每次模拟面试的追问深度 |
| 训练计划完成率 | 用户完成建议练习的比例 |

## AI 质量指标

| 指标 | 定义 |
| --- | --- |
| JD 解析准确率 | 岗位关键词和能力要求是否提取正确 |
| 题目相关性 | 问题是否贴合目标岗位 |
| 追问相关性 | 追问是否基于用户回答继续深挖 |
| 评分一致性 | 同类回答评分是否稳定 |
| 建议可执行性 | 反馈是否具体可改 |

## 评分 Rubric

- 产品结构：是否讲清用户、痛点、方案、指标、迭代。
- AI 理解：是否理解 RAG、Prompt、Agent、Eval、幻觉。
- 指标意识：是否能定义产品指标和 AI 质量指标。
- 风险意识：是否能说明安全、隐私、拒答、人审。
- 表达可信度：是否基于真实经历，不夸大。

## 上线门禁

- 题目相关性 >= 85%。
- 追问相关性 >= 80%。
- 评分一致性 >= 80%。
- 反馈可执行性人工抽检通过率 >= 85%。
