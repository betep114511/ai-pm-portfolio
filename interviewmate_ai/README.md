# InterviewMate AI 作品集项目

InterviewMate AI 是一个面向转岗 AI 产品经理求职者的面试陪练 Agent。它根据目标 JD 生成题库，进行多轮追问，并从产品思维、AI 理解、项目表达、指标意识和可信度五个维度给出反馈。

## 项目定位

这个项目适合作为补充项目，用于展示：

- Agent 多轮对话流程设计。
- 个性化题库生成。
- 回答评分 Rubric。
- 用户成长和练习闭环。

## 文件说明

- `prd.md`：完整 PRD。
- `case_study.md`：一页项目复盘。
- `metrics_and_eval.md`：指标和评测方案。
- `presentation_guide.md`：展示指南。
- `interview_script.md`：面试讲述稿。
- `data/question_bank.csv`：样例题库。
- `prototype/index.html`：静态交互原型。

## 简历一句话

独立设计 InterviewMate AI 面试陪练 Agent，支持 JD 解析、题库生成、多轮追问、回答评分和个性化训练计划，体现 Agent 流程设计和用户成长闭环。
